# LIS3DH Motion Wake-up Device

Устройство на ESP32, которое спит в deep sleep и просыпается при обнаружении движения через акселерометр LIS3DH. Также отслеживает напряжение автомобильного генератора и не засыпает пока двигатель работает.

## Логика работы

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  [Движение]                      [Запуск двигателя]            │
│      │                                   │                      │
│      ▼                                   ▼                      │
│  LIS3DH INT1 → GPIO33 HIGH    Напряжение > 11.5 В              │
│      │                                   │                      │
│      └──────────────┬────────────────────┘                      │
│                     ▼                                           │
│             ESP32 просыпается                                   │
│                     │                                           │
│          ┌──────────┴──────────┐                               │
│          ▼                     ▼                               │
│     Есть движение?     Генератор работает?                     │
│       (< 5 сек)          (> 11.5 В)                           │
│          │                     │                               │
│     ДА → не спать        ДА → не спать                        │
│     НЕТ ↘                НЕТ ↙                                │
│            Оба условия = НЕТ                                   │
│                     │                                           │
│                     ▼                                           │
│               Deep sleep                                        │
│          (до следующего INT1)                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Потребление в deep sleep:** ~20 мкА (ESP32 ~10 мкА + LIS3DH ~10 мкА)

## Схема подключения

```
ESP32                  LIS3DH
──────                 ──────
GPIO21 (SDA) ───────── SDA
GPIO22 (SCL) ───────── SCL
GPIO33       ───────── INT1     ← wake-up (RTC GPIO)
3.3V         ───────── VCC
3.3V         ───────── CS       ← обязательно (I2C режим)
GND          ───────── GND
GND          ───────── SA0      → адрес 0x18


Делитель напряжения генератора (0–18 В → 0–3.3 В):

Генератор (+) ──┤ R1=100кΩ ├── GPIO35 ──┤ R2=27кΩ ├── GND

Делитель напряжения батареи ESP32 (0–8.4 В → 0–3.3 В):

Vbat (+) ──┤ R1=100кΩ ├── GPIO34 ──┤ R2=100кΩ ├── GND
```

> **GPIO33 и GPIO35** — RTC GPIO, поддерживают ext0 wakeup и input-only ADC.
> Другие подходящие RTC GPIO: 0, 2, 4, 12–15, 25–27, 32–39.

## Настройка параметров

Все ключевые параметры находятся в секции `globals` файла `device.yaml`:

| Переменная | По умолчанию | Описание |
|------------|:------------:|----------|
| `still_timeout_ms` | `5000` | Сколько мс тишины нужно до засыпания |
| `motion_accel_threshold` | `1.47` | Порог движения в м/с² (0.15g × 9.8) |
| `generator_threshold_v` | `11.5` | Порог напряжения «генератор работает» (В) |

### Порог генератора

| Состояние | Напряжение |
|-----------|-----------|
| АКБ разряжена | < 11.8 В |
| АКБ в покое | 12.0–12.6 В |
| Двигатель работает | **13.5–14.8 В** |
| Перезаряд / неисправность | > 15.0 В |

Порог `11.5 В` выбран так, чтобы надёжно отличать работающий генератор от заряженной АКБ в покое. Если нужна более точная граница — установи `13.0`.

## Сенсоры в Home Assistant

| Сенсор | Тип | Описание |
|--------|-----|----------|
| Ускорение X / Y / Z | Число, м/с² | Данные акселерометра |
| Полное ускорение | Число, м/с² | √(X²+Y²+Z²) |
| Напряжение генератора | Число, В | АЦП GPIO35, делитель × 4.704 |
| Напряжение батареи | Число, В | АЦП GPIO34, делитель × 2.0 |
| Заряд батареи | Число, % | Линейное приближение 3.0–4.2 В |
| Генератор работает | Бинарный | `on` если напряжение > 11.5 В |
| Движение | Бинарный | `on` пока идёт движение |
| Пробуждение от движения | Бинарный | `on` если проснулся от INT1 |
| Причина пробуждения | Текст | Движение / Таймер / Включение |
